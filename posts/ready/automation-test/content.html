생활해결노트 자동 게시 테스트

이 글은 GitHub Actions와 Blogger API가 정상적으로 연결되어 자동 게시까지 수행되는지 확인하기 위한 테스트 게시글입니다. 생활해결노트의 실제 운영을 시작하기 전에 승인 과정, 파일 검증, Google 인증, Blogger 게시가 하나의 흐름으로 정상 작동하는지 확인하는 것이 목적입니다.

이번 테스트에서 확인하는 항목

  • 게시글 메타데이터 파일인 post.json이 올바르게 읽히는지 확인합니다.
  • HTML 본문 파일인 content.html이 검증 기준을 충족하는지 확인합니다.
  • GitHub Pull Request를 main 브랜치에 병합하면 게시 작업이 자동으로 실행되는지 확인합니다.
  • GitHub Secrets에 저장한 Google 인증 정보로 Blogger API에 접근할 수 있는지 확인합니다.
  • 생활해결노트 Blogger에 새 게시글이 정상적으로 공개되는지 확인합니다.

자동 게시 시스템의 목표

앞으로 생활해결노트의 글은 주제 선정, 자료 조사, 글 작성, 검색엔진 최적화, 본문 HTML 준비 과정을 거쳐 GitHub Pull Request로 등록됩니다. 사용자는 게시할 내용을 확인한 뒤 Merge 버튼으로 승인하고, 승인된 글은 GitHub Actions를 통해 Blogger에 자동 게시됩니다.

이 테스트 글이 생활해결노트 블로그에 공개되면 기본 자동 게시 시스템이 정상적으로 연결된 것입니다. 테스트 완료 후에는 실제 생활 문제 해결 콘텐츠를 같은 방식으로 작성하고 게시할 예정입니다.
